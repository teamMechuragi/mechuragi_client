# 🐘 Mechuragi
ChatGPT API 기반 메뉴 추천 커뮤니티 플랫폼

---

## 📌 프로젝트 소개

**Mechuragi**는 ChatGPT API를 활용하여 사용자의 기분, 날씨, 취향, 재료 등을 종합적으로 분석하고  
가장 적합한 메뉴를 추천하는 **AI 기반 식사 추천 커뮤니티 앱**입니다.  
기록 기능과 커뮤니티 투표 기능도 함께 제공하여 **식사 결정부터 공유, 기록까지** 한 번에 해결할 수 있습니다.

---

## 🎯 제작 목표

🍽 AI 기반 맞춤 메뉴 추천  
📅 먹방 일기 캘린더로 식사 기록  
👥 커뮤니티 투표로 메뉴 선택  
🧠 ChatGPT API를 활용한 대화형 추천 시스템 구현

---

## ✅ 기대 효과

- 사용자의 상황과 선호도에 따른 맞춤 식사 추천 제공
- 메뉴 결정 스트레스 해소
- 식사 데이터를 통한 개인화된 경험 축적
- 커뮤니티 기반 식사 공유 및 소통 경험 강화

---

## ⚙️ 주요 기능

| 구분       | 기능 설명 |
|------------|-----------|
| AI 추천   | 기분, 날씨, 재료, 시간대 기반 AI 메뉴 추천 |
| 먹방 일기 | 캘린더 기반의 식사 기록 및 사진 저장 |
| 취향 설정 | 선호 음식, 맛, 인원, 알레르기 정보 입력 |
| 커뮤니티 투표 | “오늘 뭐 먹지?”를 주제로 투표 생성 및 참여 |
| 사용자 기능 | 로그인 / 회원가입 / 설정 |
| 리뷰 기능 | 메뉴 리뷰 작성 및 별점 등록 |
